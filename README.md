# WordleCLI

This is a Kotlin project that implements a basic version of the Wordle game, with a CLI interface. The project is built
with Gradle. Refer to the following command to build the project:

```bash
./gradlew build
```

After that the targeted JAR file will be generated in the `build/libs` directory. You can run the game with the
following command:

```bash
java -jar build/libs/WordleCLI-1.0-SNAPSHOT-all.jar # Change the version number if necessary
```
